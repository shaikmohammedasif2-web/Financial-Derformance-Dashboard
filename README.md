# 📊 Financial Performance Dashboard

## 📌 Project Overview
This Power BI dashboard provides an interactive analysis of financial performance, helping businesses monitor sales, profit, expenses, and overall business growth through visual insights.

## 🎯 Objectives
- Analyze financial performance
- Track sales and profit trends
- Monitor KPIs
- Identify top-performing categories and regions

## 🛠 Tools Used
- Power BI
- Microsoft Excel
- DAX
- Power Query

## 📈 Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- Sales Growth
